# CS 81 - Module 3A GitHub Assignment

This repository demonstrates the use of Git and GitHub with JavaScript concepts.
The project includes a journal-style JavaScript file containing multiple functions, each showcasing different programming techniques.

_____________________________________________________________________________________________________________________________________

## journal.js Functions

### greet(name)
Returns a greeting message using the provided name.
_____________________________________________________________________________________________________________________________________

### howWasYourDay(response)
Uses conditional logic to respond to how the user describes their day.
Valid responses include good, okay, or bad. Invalid responsesare handled gracefully.
_____________________________________________________________________________________________________________________________________

### groceryReminderList()
Creates and returns a grocery reminder list containing common household items such as milk, diapers, and toiletry.
_____________________________________________________________________________________________________________________________________

### calculateGrade(score)
Converts a numeric percentage score into a letter grade (A-F) based on standard grading scale.
Handles invalid scores outside the 0-100 range.
_____________________________________________________________________________________________________________________________________

### sleepHoursFeedback(hours)
Provides a professional feedback based on the number of hours slept.
Includes validation for unrealistic inputs.
